{{'Improvements for Mappers'| translate }}

  • {{'Expanded User Profile and Contact information - Please make the first thing you do in the new Tasking Manager visiting your'| translate }} {{ 'user profile' | translate }}. {{'There you will see your mapper experience level and most importantly, can fill in your user contact information so you can get feedback on your mapping delivered to your email in-box.'| translate }}

  • {{'Finding a Project - When you visit the Contribute link to find a project that needs mapping, you will have a new set of options to help you find a project to work on. You can now filter by:'| translate }}

    • {{'Difficulty - Every new project will have a difficulty rating so you can work on projects that are best for your experience level.'| translate }}

    • {{'Organization - Select from the list of organizations that make use of OpenStreetMap map data'| translate }}

    • {{'Campaign - Often projects are grouped into campaigns, efforts that need multiple projects to get done, now you can easily find all the projects in a campaign.'| translate }}

    • {{'Type of Mapping - Roads, buildings, waterways, find the kind of mapping you are in the mood for.'| translate }}

  • {{'More ways to ask questions and get feedback - There is now a "Questions and Comments" tab on each project, with a quick link to direct a question to the mapping project manager.'| translate }}

{{'Improvements for Validators'| translate }}

  • {{'Expanded User Profile and Contact information - Please make the first thing you do in the new Tasking Manager visiting your '| translate }}{{ 'user profile' | translate }}. {{'The Tasking Manager now tracks and displays the number of Tasks you have validated. Since users can also enter their email addresses, when you direct mapping feedback to a user, they are much more likely to see your feedback (make sure to add your email address to your profile too.)'| translate }}

  • {{'Validate multiple tasks at once - While you can still validate one task at a time, you can also checkout multiple tasks at once to validate. You have to option of drawing a polygon around a group of tasks, or you can select all the tasks completed by an individual mapper. You can sort contributing users by:'| translate }}

    • {{'Experience level - Based on the number of changesets a user has in OSM you can get an immediate idea of how experienced a mapper is.'| translate }}

    • {{'Registration date - This tells you how long a user has been using the Tasking Manager. It will help you find brand new users who can benefit from immediate feedback.'| translate }}

    • {{'Time since last validation - This will let you know how long it has been since a user received feedback.'| translate }}

  • {{'Project level comments and full task comment feed - With the new "Questions and Comments" tab on each project and a full list of all the comments on individual tasks, you will be able to find users who have questions or identify issues with the project quickly.'| translate }}

{{'Improvements for Project Managers'| translate }}

  • {{'Split Tasks during project creation - While creating a project you can "pre-split" tasks over dense settlement areas. You can also use satellite imagery during the project creation process to help you determine good Task sizes.'| translate }}

  • {{'Project Grouping by requesting organization and campaigns - You can easily group projects by different organizations and campaigns. This allows you to easily see over all progress via your Project Dashboard in the user menu.'| translate }}

  • {{'Project level comments and full task comment feed - You can quickly review task and project level comments to identify issues and answer questions.'| translate }}

  • {{'More project permissions options - You can restrict mapping to only intermediate or experienced mappers and validators'| translate }}

  • {{'Project cloning - Easily create duplicate projects and change the AoI or the instructions'| translate }}

  • {{'Expanded User Profile and Contact information - Please make the first thing you do in the new Tasking Manager visiting your'| translate }} {{ 'user profile' | translate }} {{'and entering your email address so Tasking Manager messages sent to you will go to your email box.
